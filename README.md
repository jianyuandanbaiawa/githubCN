# GithubCN

> 维护中，已新增1837 条翻译 ，覆盖了 GitHub 上可能出现的绝大多数编程相关英文词汇！

Github 浏览器中文汉化插件

## 如何补充翻译词条？

所有的翻译内容都在在`src/js/content.js`中

```js
const allData = [
  [`English`, `英文`],
]
```
